# 🔍 שלב 8.5 — תחנת בדיקה (check-station)

- מציג הזמנות מ-`workDay.inChisum` בלבד (Firebase)
- כל הזמנה: סקיצה/רשימת פריטים + סימון פריטים כ"נבדק"
- כפתור "ייבוא ממתינות" — מופיע כשיש `pendingChisum`; ייבוא מעביר pendingChisum → inChisum

## זרימה בתחנה:
1. לוחצים על הזמנה → רואים פריטים/סקיצה
2. מסמנים פריטים שנבדקו
3. לוחצים "סיים" → updateStage(id, 'chisum') + מוסר מ-inChisum

## כשהתחנה מתרוקנת:
✔ pendingChisum עולים אוטומטית ל-inChisum
